    About the end of support for GuessGame...

    First of all, RVH Productions wants to thank all of those who have played GuessGame since it's release on December 13, 2020.
    Support for GuessGame has ended on July 1st, 2025 after nearly five years of development.

    The choice to cease development has been made because the old code base is difficult to work with, as some
    of the old code originally written in 2020 is still present in the latest development build of GuessGame.

    Continuing serious development on the game would require me to rewrite huge parts, if not all of the game.
    However, given the product's age, I've chosen to let it rest.

    Online endpoints for GuessGame 1.4, 1.5 and the 1.6 Beta will remain active until at least January 1st, 2026.


    Again, thank you for playing GuessGame. It has been an amazing ride!
    ~ Rico

    System requirements

    Operating system

    Windows 10 64-bit or later for version 1.6 and higher;
    Windows 8.1 (any architecture) or later for version 1.5 and its later revisions;
    Windows 8.1 (any architecture) or later for version 1.4 and its later revisions;
    Windows 7 (any architecture) or later for version 1.3.1 and older*.

    Device

    Must be capable of running Windows
    Internet connection required to access Store, News and Software update.

    *Windows 7 is the oldest version of Windows where these versions of GuessGame have been tested on.
