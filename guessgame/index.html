    GuessGame News

    Version 1.5.2 released

    What has changed?

    - The game no longer creates profiles with 100 Hints;
    - Some other bugs fixed.

    Support for GuessGame 1.4 is coming to an end

    Support for GuessGame 1.4 (release channel) is ending on April 15, 2023.
    The end of support date has been moved from June 7 to April 15.

    This means that after April 15, 2023, RVH Productions will no longer provide the following for GuessGame 1.4:
    - Quality updates
    - Security fixes
    - Network support

    RVH Productions recommends updating to version 1.5.2 to stay safe and to get the latest quality updates.

    System requirements

    Operating system

    Windows 10 64 bit or later for versions 1.6 and higher*;
    Windows 8.1 (any architecture) or later for versions 1.5 and its later revisions;
    Windows 8.1 (any architecture) or later for versions 1.4 and its later revisions;
    Windows 7 (any architecture) or later for versions 1.3.1 and older**.

    Device

    Must be capable of running Windows
    Must have an internet connection to access Store, News and Software update.

    *This requirement is for a version of GuessGame that has not been released yet. This requirement can be changed in the near future.

    **Windows 7 is the oldest version of Windows where these versions of GuessGame have been tested on.
