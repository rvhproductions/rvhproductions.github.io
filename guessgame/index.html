    GuessGame News

    Version 1.5 released

    What has changed?

    - Added Hints;
    - Added Store;
    - Added Time limit feature;
    - Added a "More information" button under About GuessGame in Settings;
    - Added the amount of Hints and remaining time to the in game screen;
    - Redesigned the Profile Info screen;
    - Redesigned the Credits screen, now renamed to About GuessGame, showing more useful info;
    - Exp required for level up can't be higher than 100.000.

    Bugs fixed:

    - The Show more updates button will no longer show up when there's no internet connection;
    - Fixed the amount of guesses it took to not show up correctly after guessing right.


    Version 1.5.2 released

    What has changed?

    - The game no longer creates profiles with 100 Hints;
    - Some other bugs fixed.

    Introducing GuessGame 1.6

    The next big release of GuessGame.

    With GuessGame 1.6, your playing experience becomes better than ever.
    Featuring a redesigned Main Menu, profile selection screen and a brand new gameplay feature: Powerups.

    Powerups
    Powerups can be bought in the Store. They boost your rewards after finising a game.
    There are two types of Powerups:
    EXP Powerups, which double your EXP rewards;
    GuessCoin Powerups, which double your GuessCoin rewards.

    Redesigned Main Menu
    GuessGame 1.6 also features a redesigned Main Menu.
    The Main Menu now looks better than ever before.
    Featuring new buttons, an easy way of seeing the remaining time of active Powerups,
    and an easier to use Quit selection screen.

    Redesigned profile selection screen
    The profile selection screen has also been redesigned.
    You are now greeted with a beautiful animation upon starting GuessGame.
    The buttons have also been made bigger and easier to use.

    Interested in testing? Beta versions are now available to everyone, for free.
    You can download the latest Beta here.
    Be careful!
    Beta versions of GuessGame can be buggy sometimes, and it is strongly recommended to make a backup of your profiles
    before loading them.

    I found a bug, what now?
    Consider contacting RVH Productions through our social media channels, like Twitter or Discord.

    System requirements

    Operating system

    Windows 10 64 bit or later for version 1.6 and higher;
    Windows 8.1 (any architecture) or later for version 1.5 and its later revisions;
    Windows 8.1 (any architecture) or later for version 1.4 and its later revisions;
    Windows 7 (any architecture) or later for version 1.3.1 and older*.

    Device

    Must be capable of running Windows
    Internet connection required to access Store, News and Software update.

    *Windows 7 is the oldest version of Windows where these versions of GuessGame have been tested on.
