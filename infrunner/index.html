    • Introducing InfRunner

    RVH Productions is happy to announce: InfRunner.
    This is a new game by RVH Productions. This page will tell you what InfRunner (currently) is, and what features are planned.

    Infinite running game
    InfRunner is an endless running game, where the player needs to get as much score as possible without crashing into an enemy.
    At the same time, the player needs to collect multiplier coins, which increases the score multiplier.

    Planned features
    Multiplayer! This is a feature that is currently being worked on.
    Worlds created by players! In the near future, player can make their own worlds, and can add songs to their world.
    A gamemode, called Timed! This gamemode means getting as much score as possible before a song is over. This gamemode can only be played on worlds with a song.

    Free to play!
    InfRunner will be free to play for everyone, without advertisements, no in-app purchases, nothing of that nonsense.

    System requirements

    The minimum system requirements to play InfRunner smoothly at 60 FPS are as follows*:

    CPU
    3rd Generation Intel Core i3 or AMD equivalent

    Display
    60hz or greater

    Operating System
    Microsoft Windows 10 or newer.
    InfRunner does not work on Linux.


    *These requirements may change over time. InfRunner will always attempt to run on devices that don't meet these requirements.
